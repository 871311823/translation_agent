4. 点击"📁 打开输出文件夹"查看结果

## 🔧 常见问题

### Q1: 软件启动后报错？
**A**: 清理Python缓存后重启：
```powershell
Get-ChildItem -Path . -Include __pycache__,*.pyc -Recurse -Force | Remove-Item -Recurse -Force
python translation_agent_gui.py
```

### Q2: API连接测试一直"连接中"？
**A**: 
- 检查网络连接
- 确认API服务器地址正确
- 等待最多30秒，会自动超时并显示错误
- 如果超时，检查防火墙设置

### Q3: 扫描文件报错？
**A**: 
- 确保已清理Python缓存
- 重启软件
- 检查输入文件夹路径是否正确

### Q4: 翻译结果为空？
**A**: 
- 增加max_tokens参数（建议1000以上）
- 检查源文件是否有内容
- 查看"翻译进度"页面的错误信息

### Q5: 翻译速度慢？
**A**: 
- 降低并发任务数
- 检查网络连接速度
- 考虑使用更快的API端点（如Groq）

## 💡 使用技巧

### 1. 批量处理大量文件
- 将文件按大小分组
- 小文件使用高并发（8-10）
- 大文件使用低并发（2-3）

### 2. 保存常用配置
- 配置好API后点击"💾 保存配置"
- 下次启动自动加载配置
- 无需重复输入

### 3. 使用快速配置预设
- 点击右侧的快速配置按钮
- 一键应用OpenAI、Groq或Ollama配置
- 只需填写API密钥即可

### 4. 监控翻译进度
- 实时查看每个文件的翻译状态
- 点击任务查看详细信息
- 查看错误信息快速定位问题

### 5. 输出格式选择
- **TXT格式**: 
  - 优点：兼容性好，文件小
  - 适合：纯文本翻译，批量处理
- **DOCX格式**: 
  - 优点：格式丰富，专业美观
  - 适合：正式文档，需要编辑的内容

## 📊 性能优化

### 网络优化
- 使用稳定的网络连接
- 避免使用VPN（除非必要）
- 选择地理位置近的API服务器

### 参数优化
- 根据文本长度调整max_tokens
- 短文本：500 tokens
- 中等文本：1000 tokens
- 长文本：2000 tokens

### 并发优化
- 根据API限制调整RPM
- 根据文件大小调整并发数
- 监控系统资源使用情况
